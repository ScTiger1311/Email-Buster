Content Creation

How to add more emails

The following guide makes no assumptions about whatever form of version control you are using, as I currently don't know. Make sure you have a copy of the source code, and a way to merge any changes you might make.
This guide may seem intimidating, but I tried to document things very clearly. It should take no more than ~8 minutes per email assuming you know what to write, and understand the workflow
  1. If you haven't already, download the Emails and Templates Master Files zip folder.
  2. The two files you need are located at
    1. .\Emails\Template\Mail.psd
    2. .\Emails\Template\Preview.psd
  3. Open Mail.psd in Photoshop
    1. Select Save As, and depending on if you want it to be a fake or real email, name it either FakeEmail_XXX.psd or RealEmail_XXX.psd, where XXX is replaced with any currently unused number.
    2. Edit the text however you like.
    3. You can also add images if you really want to.
    4. When you're done save the .psd, then select File->Export->Export As... and export it as a PNG with the same name as the psd. Make all layers except the email content are hidden in the preview, the background should be transparent.
  4. Open Preview.psd in Photoshop.
    1. Select Save As, and depending on if you want it to be a fake or real email, name it either FakeEmail_Preview_XXX.psd or RealEmail_Preview_XXX.psd where XXX is replaced with the corresponding number of the email the preview should be associated with.
    2. Edit the text however you like
    3. When you're done save the .psd, then select File->Export->Export As... and export it as a PNG with the same name as the psd. Make all layers except the email content are hidden in the preview, the background should be transparent.
  5. Copy both the FakeEmail_XXX.png and FakeEmail_Preview_XXX.png files to \assets\single_sprites\Mail ,which is part of the source code for the game. If there's already files with that name there, don't select replace. Give them a new unique name, usually it should just be 1 higher than the highest number mail.
  6. Using a code editor of your choice, open \src\scenes\Play.js, located in the source code of the game.
    There should be a portion of code that looks like this near the beginning of the file. Notice that there is an array called this.unusedMailReal, and an array called this.unusedMailFake. In that is a bunch of lines that start with new Mail(...), Copy one of those lines from the correct fake/real array, and paste it at the bottom of the list, before the close bracket. Update the two file paths in quotation marks to the file paths of the new email, and preview, respectively. If you've been following the guide so far, you should just be able to update the number at the end of each of them. If you are adding a fake email, make sure to update the final 4 true/false things appropriately.
    1. Below are what the 4 true/false values correspond to, in order.
      1. Sender is posing as someone they aren't
      2. Email contains link to a suspicious website
      3. Email requests personal info
      4. Email contains suspicious attachment.
  7. When you are done, save Play.js, and merge using whatever version control system you are using.
